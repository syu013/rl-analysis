# rl-analysis 
ゲームで遊んだプレイヤーの行動解析を行うWebアプリケーションです。  
遊んだプレイヤーの操作履歴からどのような行動特徴を持っているのか、プレイヤー自身が見ることができます。  
これにより、プレイヤーは自身のプレイを簡単に分析できるため、ゲームプレイの上達を手助けできます。  

## App URL
https://rl-analysis.herokuapp.com/
